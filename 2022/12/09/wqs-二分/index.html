# 问题描述

wqs\text{wqs}wqs 二分解决的问题有这样的特性：

  • 有一堆物品，题目要求你恰好选择 kkk 个物品使代价最大 / 最小。

  • 如果不限制物品个数，问题可以很容易的解决（指复杂度比限制了要更低）。

  • 设 g(i)g\left(i\right)g(i) 表示选 iii 个物品的最大 / 最小代价，则 g(i)g\left(i\right)g(i) 关于 iii 的函数是个凸单调的函数。

如果一个问题符合上述特征，那么就可以用 wqs\text{wqs}wqs 二分来对这个问题降维 / 降复杂度处理了。

# 算法流程

参考一下下面这个函数 ggg：

现在我们知道了 ggg 的大致轮廓，但是我们不知道每个点具体的函数值。

但是根据凸函数的性质，它的斜率是单调的，我们考虑二分这个斜率。假设我们二分的这个斜率是 kkk，我们用这个斜率为 kkk 的直线去和这个凸包相切。可以发现，随着 kkk 的减小，这个直线所切的点会越来越靠右。如下图所示：

现在我们假设直线切的点是 (x,g(x))\left(x,g\left(x\right)\right)(x,g(x))，那么很显然直线和 yyy 轴的截距是 g(x)−kxg\left(x\right)-kxg(x)−kx。我们记为 f(x)f\left(x\right)f(x)。那么它的具体含义是什么呢？相当于我们把选的 xxx 个物品代价都减去一个 kkk，那么截距的含义就是在减去代价的基础上的花费。

考虑下面这张图：

在用同样斜率的直线去经过凸包上的每个点的时候都会产生一个截距，截距表示选择 xxx 个物品的基础上，在代价减小的时候取得的价值。切点对应着最高的 fff 值。最高的 fff 值什么意思？在不限制物品个数、给物品减去代价的时候取得的最大代价！问题转化为了没有物品数量限制的问题，很好解决了。

所以流程就是：

  1. 二分直线的斜率 kkk。

  2. 对于每个 kkk 都去解决一个不限个数的问题，计算最大的 fff 值的同时还计算了这个 fff 对应的 xxx，从而通过 f(x)+kxf\left(x\right)+kxf(x)+kx 求出 g(x)g\left(x\right)g(x)。如果 xxx 不满足个数要求，则可以用二分缩小区间，尝试下一个斜率 kkk，直到满足个数要求。

在做的过程中，还有很多细节需要注意，比如一条切线切在多个点的情况（这种情况通常在我们求不限个数问题时解决，对于相同代价，我们强制选最多 / 最少的物品，然后对应去修改我们二分部分的区间取舍），等等。我们通过例题来解决这些问题。
