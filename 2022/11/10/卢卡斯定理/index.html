# 定理内容

卢卡斯定理用来求解大组合数对素数取模的问题，其内容如下：

对于素数 ppp，有 (nm)modp≡(⌊n/p⌋⌊m/p⌋)(nmodpmmodp)(modp)\binom{n}{m}\bmod p\equiv\binom{\left\lfloor n/p\right\rfloor}{\left\lfloor m/p\right\rfloor}\binom{n\bmod p}{m\bmod p}\pmod p(mn​)modp≡(⌊m/p⌋⌊n/p⌋​)(mmodpnmodp​)(modp)。

观察上述表达式，可知 nmodpn\bmod pnmodp 和 mmodpm\bmod pmmodp 一定是小于 ppp 的数，可以直接求解，(⌊n/p⌋⌊m/p⌋)\displaystyle\binom{\left\lfloor n/p \right\rfloor}{\left\lfloor m/p\right\rfloor}(⌊m/p⌋⌊n/p⌋​) 可以继续用卢卡斯定理求解。这也就要求 ppp 的范围不能够太大，一般在 10510^5105 左右。边界条件：当 m=0m=0m=0 的时候，返回 111。

时间复杂度为 O(f(p)+g(n)log⁡n)O(f(p) + g(n)\log n)O(f(p)+g(n)logn)，其中 f(n)f(n)f(n) 为预处理组合数的复杂度，g(n)g(n)g(n) 为单次求组合数的复杂度。

# 定理证明

考虑 (pn)modp\displaystyle\binom{p}{n} \bmod p(np​)modp 的取值，注意到 (pn)=p!n!(p−n)!\displaystyle\binom{p}{n} = \frac{p!}{n!(p-n)!}(np​)=n!(p−n)!p!​，分子的质因子分解中 ppp 的次数恰好为 111，因此只有当 n=0n = 0n=0 或 n=pn = pn=p 的时候 n!(p−n)!n!(p-n)!n!(p−n)! 的质因子分解中含有 ppp，因此 (pn)modp=[n=0∨n=p]\displaystyle\binom{p}{n} \bmod p = [n = 0 \vee n = p](np​)modp=[n=0∨n=p]。进而我们可以得出

(a+b)p=∑n=0p(pn)anbp−n≡∑n=0p[n=0∨n=p]anbp−n≡ap+bp(modp)\begin{aligned} (a+b)^p &= \sum_{n=0}^p \binom pn a^n b^{p-n}\\ &\equiv \sum_{n=0}^p [n=0\vee n=p] a^n b^{p-n}\\ &\equiv a^p + b^p \pmod p \end{aligned}(a+b)p​=n=0∑p​(np​)anbp−n≡n=0∑p​[n=0∨n=p]anbp−n≡ap+bp(modp)​

注意过程中没有用到费马小定理，因此这一推导不仅适用于整数，亦适用于二项多项式。因此我们可以考虑二项式 fp(x)=(axn+bxm)pmodpf^p(x)=(ax^n + bx^m)^p \bmod pfp(x)=(axn+bxm)pmodp 的结果

fp(x)≡f(xp)\begin{aligned} f^p\left(x\right)\equiv f\left(x^p\right) \end{aligned}fp(x)≡f(xp)​

考虑二项式 (1+x)nmodp(1+x)^n \bmod p(1+x)nmodp，那么 (nm)\displaystyle\binom n m(mn​) 就是求其在 xmx^mxm 次项的取值。使用上述引理，我们可以得到

(1+x)n≡(1+x)p⌊n/p⌋(1+x)nmodp≡(1+xp)⌊n/p⌋(1+x)nmodp\begin{aligned} (1+x)^n &\equiv (1+x)^{p\lfloor n/p \rfloor} (1+x)^{n\bmod p}\\ &\equiv (1+x^p)^{\lfloor n/p \rfloor} (1+x)^{n\bmod p} \end{aligned}(1+x)n​≡(1+x)p⌊n/p⌋(1+x)nmodp≡(1+xp)⌊n/p⌋(1+x)nmodp​

注意前者只有在 ppp 的倍数位置才有取值，而后者最高次项为 nmodp≤p−1n\bmod p \le p-1nmodp≤p−1，因此这两部分的卷积在任何一个位置只有最多一种方式贡献取值，即在前者部分取 ppp 的倍数次项，后者部分取剩余项，即 (nm)modp=(⌊n/p⌋⌊m/p⌋)⋅(nmodpmmodp)modp\displaystyle\binom{n}{m}\bmod p = \binom{\left\lfloor n/p \right\rfloor}{\left\lfloor m/p\right\rfloor}\cdot\binom{n\bmod p}{m\bmod p}\bmod p(mn​)modp=(⌊m/p⌋⌊n/p⌋​)⋅(mmodpnmodp​)modp。
