# 求连续 n 个数的逆元

问题描述：求出 1,2,⋯,n1,2,\cdots,n1,2,⋯,n 关于 ppp 的逆元。

首先我们应该注意到 1−1≡1(modp)1^{-1}\equiv1\pmod p1−1≡1(modp)。

考虑递推求 i−1i^{-1}i−1。我们令 k=⌊pi⌋,j=pmodik=\left\lfloor\frac{p}{i}\right\rfloor,j=p\bmod ik=⌊ip​⌋,j=pmodi，显然有 ki+j≡0(modp)ki+j\equiv0\pmod pki+j≡0(modp)。

两边同乘 i−1×j−1i^{-1}\times j^{-1}i−1×j−1，得：

kj−1+i−1≡0(modp)i−1≡−kj−1(modp)i−1≡−⌊pi⌋(pmodi)−1(modp)\begin{aligned} kj^{-1}+i^{-1}&\equiv0&\pmod p\\ i^{-1}&\equiv-kj^{-1}&\pmod p\\ i^{-1}&\equiv-\left\lfloor\frac{p}{i}\right\rfloor\left(p\bmod i\right)^{-1}&\pmod p \end{aligned}kj−1+i−1i−1i−1​≡0≡−kj−1≡−⌊ip​⌋(pmodi)−1​(modp)(modp)(modp)​

# 线性求任意 n 个数的逆元

现在我们求的不是 1⋯n1\cdots n1⋯n 的逆元了，而是任意 nnn 个数，怎么办呢？

首先考虑计算出 nnn 个数的前缀积，记为 sns_nsn​，然后使用快速幂或是扩展欧几里得求出 sns_nsn​ 的逆元，记为 svnsv_nsvn​。然后我们每次将 svisv_isvi​ 乘上 aia_iai​，即可求出 svi−1sv_{i-1}svi−1​。而 aia_iai​ 的逆元则由 svi×si−1sv_i\times s_{i-1}svi​×si−1​ 得来。

时间复杂度是 O(n+log⁡p)O\left(n+\log p\right)O(n+logp)。
