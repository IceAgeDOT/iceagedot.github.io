# 问题概述

一个整数 aaa，如果不是 ppp 的倍数且模 ppp 同余于某个数的平方，则称 aaa 为模 ppp 的 二次剩余。而一个不是 ppp 的倍数的整数 bbb 不同余任何数的平方，则称 bbb 为模 ppp 的 二次非剩余。

对二次剩余求解，也就是对常数 aaa 解下面的这个方程：

x2≡a(modp)x^2\equiv a\pmod px2≡a(modp)

通俗一些，可以认为是求模意义下的开方，这里只讨论 ppp 为奇素数 的求解方法。

后文在模 ppp 显然的情况下简写成二次（非）剩余。

# Cipolla 算法

对于二次剩余解方程 x2≡nx^2 \equiv nx2≡n。

找到一个 aaa 满足 a2−na^2 - na2−n 是非二次剩余，由于非二次剩余的数量接近 p2\frac{p}{2}2p​，通过随机 + 检验的方式期望约 222 次可以找到这样一个 aaa。

接下来定义 i2≡a2−ni^2 \equiv a^2 - ni2≡a2−n。但是 a2−na^2 - na2−n 不是二次剩余，怎么找得到这样一个 iii？

类比实数域到复数域的推广，定义这样一个 iii，然后可以将所有数表示为 A+BiA+BiA+Bi 的形式，其中 A,BA,BA,B 都是模 ppp 意义下的数，类似于实部和虚部。

那么 (a+i)p+1≡n(a + i)^{p+1} \equiv n(a+i)p+1≡n，考虑证明。

  • 证明

    引理 111：ip≡−ii^p \equiv -iip≡−i

    • 证明

      ip≡i(i2)p−12≡i(a2−n)p−12≡−ii^p \equiv i(i^2)^{\frac{p-1}{2}} \equiv i(a^2 - n)^{\frac{p-1}{2}} \equiv -iip≡i(i2)2p−1​≡i(a2−n)2p−1​≡−i。

    引理 222：(A+B)p≡Ap+Bp(A + B)^p \equiv A^p + B^p(A+B)p≡Ap+Bp

    • 证明

      二项式定理展开后，由于 ppp 是质数，除了 Cp0,CppC_p^0,C_p^pCp0​,Cpp​ 外的组合数分子上的阶乘没法消掉，模 ppp 都会为 000，剩下来的就是 Ap+BpA^p + B^pAp+Bp。

    现在证明上述结论：

    (a+i)p+1≡(ap+ip)(a+i)≡(a−i)(a+i)≡a2−i2≡n(a + i)^{p+1} \equiv (a^p + i^p) (a + i) \equiv (a - i) (a + i) \equiv a^2 - i^2 \equiv n(a+i)p+1≡(ap+ip)(a+i)≡(a−i)(a+i)≡a2−i2≡n

    那么 (a+i)p+12(a + i)^{\frac{p+1}{2}}(a+i)2p+1​ 即是一个解，其相反数是另一个解。

    然而还剩最后一个问题，(a+i)p+12(a + i)^{\frac{p+1}{2}}(a+i)2p+1​ 的 “虚部” 一定为 000 吗？

    幸运的是，的确如此，假设存在 (A+Bi)2≡n(A + Bi)^2 \equiv n(A+Bi)2≡n 且 B≠0B \ne 0B=0，那么有 A2+B2i2+2ABi≡nA^2 + B^2i^2 + 2ABi \equiv nA2+B2i2+2ABi≡n，即 A2+B2(a2−n)−n≡−2ABiA^2 + B^2(a^2 - n) - n \equiv -2ABiA2+B2(a2−n)−n≡−2ABi。

    式子的左边 “虚部” 为 000，那么式子右边的虚部也一定为 000，也就是说 AB≡0AB \equiv 0AB≡0。既然假设了 B≠0B \ne 0B=0 那么一定是 A≡0A \equiv 0A≡0，也就是说 (Bi)2≡n(Bi)^2 \equiv n(Bi)2≡n。

    也就是 i2≡nB−2i^2 \equiv nB^{-2}i2≡nB−2，由于 B2B^2B2 是个二次剩余，其逆元 B−2B^{-2}B−2 一定也是二次剩余，乘上二次剩余 nnn 后一定还是二次剩余，这与 i2i^2i2 是个二次非剩余产生矛盾。
