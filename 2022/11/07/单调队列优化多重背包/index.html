# 问题描述

多重背包是一个这样的问题：

你有一个 nnn 个物品和容积为 VVV 的背包，每个物品的体积为 viv_ivi​，价值为 cic_ici​，最多可以选 kik_iki​ 个，求你能获得的最大价值。

# 朴素做法

我们定义 fi,jf_{i,j}fi,j​ 表示在前 iii 个物品中选，体积不超过 jjj 的最大价值，容易得到状态转移方程：

fi,j=min⁡k=0ki{fi−1,i−k×vi+k×ci}f_{i,j}=\min\limits_{k=0}^{k_i}\left\{f_{i-1,i-k\times v_i}+k\times c_i\right\}fi,j​=k=0minki​​{fi−1,i−k×vi​​+k×ci​}

可以发现时间复杂度为 O(nV∑ki)O\left(nV\sum k_i\right)O(nV∑ki​)。

分析一下时间复杂度：第一维枚举物品 iii，第二维枚举余数，第三维滑动窗口，所以复杂度是 O(nviVvi)=O(nV)O\left(nv_i\frac{V}{v_i}\right)=O\left(nV\right)O(nvi​vi​V​)=O(nV)。
