# 算法 1：试除法判定素数 + 伯特兰 - 切比雪夫定理

考虑当 k+1k+1k+1 是素数的时候，如果 k+1>n+12k+1>\frac{n+1}{2}k+1>2n+1​，那么很显然 k+1k+1k+1 和 [2,n+1]\left[2,n+1\right][2,n+1] 的所有数都互质，答案是 111；否则，根据伯特兰 - 切比雪夫定理，显然在 (n+12,n+1)\left(\frac{n+1}{2},n+1\right)(2n+1​,n+1) 内存在一个素数 ppp，它显然和 [2,n+1]\left[2,n+1\right][2,n+1] 的所有数都互质，包括 k+1k+1k+1。所以答案是 222。

如果 k+1k+1k+1 不是素数，显然一天不能使所有人都知道。但是再根据伯特兰 - 切比雪夫定理，又能找到一个素数 ppp 和所有数互质，所以答案是 222。

判定素数直接使用试除法即可，时间复杂度为 O(n)O\left(\sqrt n\right)O(n​)。
