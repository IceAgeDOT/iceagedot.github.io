# 题目描述

原题链接

给定一张有向图，请回答以下两个问题：

  1. 求出一个最小的点集，满足从这个点集出发可以遍历到这张有向图的所有点。输出点集大小。

  2. 请添加最少的边，使得这张图变成一张强连通图。输出添加的最少边数。

# 算法 1：强连通分量 + 缩点

有向图的连通性问题，考虑通过强连通分量 + 缩点，在不改变连通性的前提下将图改成拓扑图。

在拓扑图上，对于第 111 问是很好做的，直接求出所有入度为 000 的点，它们就构成最小点集。

对于第 222 问我们需要知道一个结论。我们设 cntcntcnt 表示强连通分量的数量（拓扑图的点数），srcsrcsrc 代表所有入度为 000 点（拓扑图的起点）构成的集合，dstdstdst 代表所有出度为 000 点（拓扑图的终点）构成的集合，那么：

  1. 若 cnt=1cnt=1cnt=1，那么不需要连新的边，答案为 000。

  2. 若 cnt>1cnt>1cnt>1，那么答案为 max⁡(∣src∣,∣dst∣)\max\left(\left|src\right|,\left|dst\right|\right)max(∣src∣,∣dst∣)

证明：

结论 111 显然易证，下面我们来证明结论 222。分以下两种情况讨论：

  1. ∣src∣≤∣dst∣\left|src\right|\le\left|dst\right|∣src∣≤∣dst∣

    1. 若 ∣src∣=1\left|src\right|=1∣src∣=1，只有一个起点的时候，那么我们只需要对于 dstdstdst 中的每个点向 srcsrcsrc 连一条边，那么这张图就强连通了，所连的边数是 ∣dst∣\left|dst\right|∣dst∣。

    2. 否则 ∣src∣≥2\left|src\right|\ge2∣src∣≥2，则至少有两个起点 p1,p2p_1,p_2p1​,p2​ 和两个终点 q1,q2q_1,q_2q1​,q2​，满足 p1p_1p1​ 能走到 q1q_1q1​，p2p_2p2​ 能走到 q2q_2q2​。那么我们可以从 q1q_1q1​ 向 p2p_2p2​ 连边，这样起点和终点都会减少 111。因此只要以这种方式，连接新边 ∣src∣−1\left|src\right|-1∣src∣−1 条，就转化成情况 111 了，因此答案仍是 ∣dst∣\left|dst\right|∣dst∣。

  2. ∣src∣≥∣dst∣\left|src\right|\ge\left|dst\right|∣src∣≥∣dst∣，和情况 111 情况对称，同理可证。

得证。
