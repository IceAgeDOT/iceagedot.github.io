# 基环树的基本概念

基环树 ：一张由 nnn 个点和 nnn 条边组成的连通图被称为基环树。顾名思义，基环树就是在树的基础上增添了一条边，使之形成了一个简单环。如果不保证连通，则 可能 是 基环树森林 。

内向树 ：对于一棵有向基环树，如果每个点 有且仅有一条出边 ，我们把它称之为内向树。形象化的理解，整棵树就好像以 “基环” 为中心，所有边都向内指向基环。如图所示：

为了方便之后的找环，推荐存下每个点的出边（即之后的 fafafa 数组），因为最终必定会相遇在环上。

外向树 ：对于一棵有向基环树，如果每个点 有且仅有一条入边 ，我们把它称之为外向树。形象化的理解，整棵树就好像以 “基环” 为中心，所有边都向外指出。如图所示：

为了方便之后的 DP\text{DP}DP，推荐把基环树建成 外向树 或 无向基环树 。

# 基环树 DP 的流程

基环树 DP\text{DP}DP 通常考虑划归到树形 DP\text{DP}DP 问题。有两种方法可以实现。下面给出基环树 DP\text{DP}DP 的算法流程：

  1. 在基环树上进行 DFS\text{DFS}DFS，找到环。

  2. 将基环树 DP\text{DP}DP 化为树形 DP\text{DP}DP 问题。具体而言有两种方法：

    1. 从环上每个点出发，向下做树形 DP\text{DP}DP，然后再把环纳入考虑范围内。这种方法通常适用于环上答案可以快速 O(1)O\left(1\right)O(1) 算出的情况，比如求基环树的直径。

    2. 采取环形 DP\text{DP}DP 的思路，断开环上任意一条边，做两次树形 DP\text{DP}DP，一次断开（有些题目可能不能断开，必须强制连接，比如我们的例题 111），一次强制连接（通过适当的条件和赋值实现）

  3. 得到答案。

但不管怎么说，我们首先要会在树上解决这个 DP\text{DP}DP 问题。所以在基环树开始之前，我们要先考虑清楚树上 DP\text{DP}DP 怎么解决。
