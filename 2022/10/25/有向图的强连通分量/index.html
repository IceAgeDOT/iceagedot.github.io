# 定义

强连通：有向图的强连通指在这张有向图中，每个点都能走到另一个节点。

强连通分量：有向图的强连通分量是指这张图的 极大 强连通子图。

下面介绍如何求有向图的强连通分量。

# Tarjan 算法求有向图强连通分量

# DFS 生成树

我们以以下有向图为例介绍有向图的 DFS\text{DFS}DFS 生成树：

在这张图中总共有四种边：

  1. 树边（tree edge\text{tree edge}tree edge）：示意图中以黑色边表示，每次搜索找到一个还没有访问过的结点的时候就形成了一条树边。

  2. 反祖边（back edge\text{back edge}back edge）：示意图中以红色边表示（即 7→17\rightarrow17→1），也被叫做回边，即指向祖先结点的边。

  3. 横叉边（cross edge\text{cross edge}cross edge）：示意图中以蓝色边表示（即 9→79\rightarrow79→7），它主要是在搜索的时候遇到了一个已经访问过的结点，但是这个结点 并不是 当前结点的祖先。

  4. 前向边（forward edge\text{forward edge}forward edge）：示意图中以绿色边表示（即 3→63\rightarrow63→6），它是在搜索的时候遇到子树中的结点的时候形成的。

我们考虑 DFS\text{DFS}DFS 树和强连通分量之间的关系：若 uuu 是某个强连通分量里第一个被搜索到的节点，那么它所在强连通分量的其它点 一定 在以 uuu 为根的子树中，uuu 被称为该强连通分量的根。

证明：用反证法。假设 vvv 在 uuu 所在强连通分量但不在以 uuu 为根的子树中，那么必然存在一条路径从 u→vu\rightarrow vu→v。但是这样的路径中的边只能是 反祖边 或 横叉边，然而这两种边指向的点都是已访问过的，与 “ uuu 是某个强连通分量里第一个被搜索到的节点” 矛盾。

# Tarjan 算法求强连通分量

Tarjan\text{Tarjan}Tarjan 算法为每个点 uuu 维护了以下两个变量：

  1. dfnudfn_udfnu​：节点 uuu 的 DFS\text{DFS}DFS 序。

  2. lowulow_ulowu​：从节点 uuu 出发，能走到的在栈中的 dfndfndfn 最小的点的 dfndfndfn 序。

容易发现这两个变量的单调性：

  1. 一个节点子树内所有点的 dfndfndfn 都比这个点的 dfndfndfn 要大。

  2. 从根开始的一条路径的 dfndfndfn 严格递增，lowlowlow 严格不降。

按照深度优先搜索算法搜索的次序对图中所有的结点进行搜索，维护每个结点的 dfndfndfn 与 lowlowlow 变量，且让搜索到的结点入栈。每当找到一个强连通元素，就按照该元素包含结点数目让栈中元素出栈。在搜索过程中，对于结点 uuu 和与其相邻的结点 vvv（vvv 不是 uuu 的父节点）考虑 333 种情况：

  1. vvv 未被访问：继续对 vvv 进行深度搜索。在回溯过程中，用 lowvlow_vlowv​ 更新 lowulow_ulowu​。因为存在从 uuu 到 vvv 的直接路径，所以 vvv 能够回溯到的已经在栈中的结点，uuu 也一定能够回溯到。

  2. vvv 被访问过，已经在栈中：根据 lowlowlow 值的定义，用 dfnv\textit{dfn}_vdfnv​ 更新 lowu\textit{low}_ulowu​。

  3. vvv 被访问过，已不在栈中：说明 vvv 已搜索完毕，其所在连通分量已被处理，所以不用对其做操作。

考虑对于一个强连通分量，这个强连通分量里面一定只有 恰好 一个点满足 dfnu=lowudfn_u=low_udfnu​=lowu​。该结点一定是在深度遍历的过程中，该连通分量中第一个被访问过的结点，因为它的 dfndfndfn 和 lowlowlow 值最小，不会被该连通分量中的其他结点所影响。

因此，在回溯的过程中，判定 dfnu=lowudfn_u=low_udfnu​=lowu​ 是否成立，如果成立，则栈中 uuu 及其上方的结点构成一个强连通分量。

# 缩点

我们考虑把每个强连通分量缩成一个点，原先点与点之间的边转变成强连通分量和强连通分量之间的边。可以证明，这样缩点之后连出来的新图是一张拓扑图（即有向无环图，如果有环，说明还有一个强连通分量没有缩点）。

而拓扑图具有很好的性质，我们可以很方便的进行 DP\text{DP}DP / 记忆化搜索，因为它不具有后效性。而原图的连通性和拓扑图是完全等价的（从强连通分量的定义中很容易得到这个结论），所以很多连通性问题都可以考虑用强连通分量 + 缩点进行等效转化。
